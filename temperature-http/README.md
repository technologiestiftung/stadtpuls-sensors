## Prerequisites

- PlatformIO - https://platformio.org/
## Installation

```bash
pip install platformio
# or
brew install platformio
```

## Usage

  
  ```bash
  pio pkg install
  pio run
  pio run --target upload
  pio device monitor -b 115200
  ```
